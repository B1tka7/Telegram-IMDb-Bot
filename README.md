# Telegram-IMDb-Films-Description

A Telegram bot that searches for movies by title and returns key information such as the description, rating, and official poster.

## Features
- Search movies by title  
- Get description and IMDb rating  
- Display the official movie poster  

## How to run
1. Install dependencies:

pip install -r requirements.txt

2. Create a `.env` file:

token=your_telegram_bot_token
apikey=your_api_key

3. Start the bot:

python main.py


## Tech stack
- Python  
- telebot  
- OMDb API
